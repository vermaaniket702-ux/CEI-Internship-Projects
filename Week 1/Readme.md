## Data Exploration and Cleaning using Pandas

### Project Overview
This project demonstrates the fundamentals of data exploration and data cleaning using Python and Pandas. The objective is to load a CSV dataset, inspect its structure, handle missing values, remove duplicate records, perform basic data operations, and export a cleaned dataset for further analysis.

### Project Structure
- Combined_dataset.csv
- Cleaned_dataset.csv
- data_exploration_cleaning.ipynb
- README.md

### Tasks performed
#### Dataset Information
- Displayed the first five rows using head()
- Displayed the last five rows using tail()
- Checked dataset dimensions using shape
- Listed all column names
- Examined data types of each column
#### Missing Value Analysis
- Identified missing values using isnull().sum()
- Filled numerical missing values using median values
- Filled categorical missing values using mode values
#### Data Cleaning
- Removed duplicate records
- Verified that no missing values remained after cleaning
#### Basic Data Operations
- Selected specific columns
- Filtered rows based on conditions
- Derieving new columns
